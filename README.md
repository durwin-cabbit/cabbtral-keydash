# Cabbtral Beta | Key Management Backend

A sleek, premium PHP backend and dashboard for managing access keys.

## Features
- **Key Generation**: Generates keys in the format `cabbtral-beta-[random_string]`.
- **Management Dashboard**: View, filter, and delete keys.
- **Copy to Clipboard**: Quick copy functionality for easy distribution.
- **Status Toggling**: Enable/Disable keys with a single click.
- **SQLite Support**: Zero-configuration database out of the box.

## Setup
1. **Host the files**: Upload all files to your PHP-enabled server (Apache/Nginx).
2. **Permissions**: Ensure the directory is writable by the web server (to create/update `database.sqlite`).
3. **Access**: Navigate to `index.php` in your browser.

## Tech Stack
- **Backend**: PHP 7.4+
- **Database**: SQLite (built-in)
- **Frontend**: Vanilla JS, CSS3 (Glassmorphism), Lucide Icons, Google Fonts (Outfit)

## Key Format
Example: `cabbtral-beta-a9F3kQxM7R2`
